**Version:** 1.0  
**Effective Date:** 2026-05-23

## Overview

This repository contains the Standard Operating Procedure (SOP) for the Fabric Department. The SOP defines the end-to-end fabric handling process including:

- Order intake
- Pre-inward verification
- Inhouse fabric processing
- Out-of-house tracking
- Quality assurance requirements

## Contents

- `index.html` - Complete Fabric Department SOP document with styling

## Purpose

To establish a standardized process for fabric receipt, verification, inspection, storage, and reporting in the Fabric Department, ensuring consistent quality and compliance with customer requirements.

## Scope

This procedure applies to all fabric materials received, inspected, and processed by the Fabric Department, including:
- Woven fabrics
- Knit fabrics
- Elastic materials
- Collar pieces
- Associated accessories

## Key Sections

1. **Purpose** - Objectives of the SOP
2. **Scope** - Applicable materials and processes
3. **Responsibilities** - Department roles and duties
4. **Fabric Order Process** - Order intake and approval workflow
5. **Pre-Inward Verification** - Document and condition checks
6. **Fabric Inward Process** - Receiving procedures
7. **Inhouse Fabric Handling** - Processing steps
8. **Quality Inspection** - 4-point inspection standards
9. **Documentation and Reporting** - Record-keeping requirements
